Current Event

Shows what is currently live on the overlay. Auto-updates when a new session starts. You can edit it manually at any time.

Common Scenarios

Normal scheduled session

No action needed. The overlay automatically shows when the session starts and hides when it ends.

Hide overlay mid-session (e.g. break)

Click HIDE OVERLAY. Click SHOW OVERLAY to bring it back. The next scheduled session will auto-reset visibility when it starts.

Fix typo while live

Edit the name or title in the Current Event section → click SAVE CHANGES. Overlay updates instantly without hiding.

Surprise / unscheduled speaker

Type name and title in Current Event section → click SHOW OVERLAY. The overlay will stay on until you hide it or the next scheduled session starts.

Speaker starts early

Find the session in the schedule list → click Jump to this speaker. Overlay shows immediately.

New day / new schedule

Click Reset Schedule to clear old data, then upload a new CSV file for the day.
