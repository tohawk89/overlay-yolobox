How to Use — Stream Control

Back

This page controls the nama penceramah banner that appears at the bottom of your live stream on YoloBox or OBS. Once your schedule is uploaded, everything runs on its own — this guide will help you handle any situation that comes up during a live broadcast.

The banner always stays visible on stream. Only the text inside it changes — automatically based on the schedule, or manually from this panel.

Your YoloBox or OBS displays the overlay page as a browser source. It updates automatically — you do not need to touch it during the live stream.

When a session's start time arrives, the banner automatically shows the program name and speaker. When the session ends, the banner stays on screen showing the last speaker for up to 2 hours, then the text clears until the next session begins.

3

Use this control panel only when you need to step in — such as fixing a name or showing a speaker that is not in the schedule.

What to Do When...

✅ Normal session — nothing to do

The banner text updates automatically based on the schedule. The banner itself always stays on screen. No action needed.

⏳ Session just ended — banner still showing

This is normal. After a session ends, the last speaker's info stays on the banner for up to 2 hours. After 2 hours, the text clears on its own. If the next session starts before then, it will switch automatically.

✏️ Speaker name has a typo

Edit the name in the On Overlay section → tap SAVE CHANGES. The banner updates on screen immediately without flashing.

⚡ Speaker starts earlier than scheduled

Find the session card under Next Event: in the list → tap Jump to this speaker. The banner switches immediately.

🎤 Surprise or unscheduled speaker

Type the program name and speaker name in the On Overlay section → tap SAVE CHANGES. The banner will show the new name immediately and stay until the next scheduled session takes over automatically.

📅 New day or new event

Tap Reset Schedule to clear the old schedule, then upload a new CSV file for the day.

Upload Schedule (CSV)

Before the event begins, upload a schedule file so the banner knows when to show each speaker automatically.

1

Prepare a .csv spreadsheet (Microsoft Excel or Google Sheets → Save As CSV) with the columns below.

2

Tap the file upload area and select your CSV. The schedule will load immediately.

3

Rows do not need to be in time order — the system sorts them automatically.

Spreadsheet columns required:

Column name What to fill in Example
SubjectProgram / kuliah name (shown on white bar)Kuliah Maghrib
DescriptionSpeaker name (shown on black bar)Ustaz Ahmad Fauzi
Start DateDate the session starts07/03/2026
Start TimeTime the session starts (24-hour)20:00:00
End DateDate the session ends07/03/2026
End-TimeTime the session ends (24-hour)21:30:00
After uploading, the session list will appear. Past sessions are removed from the list automatically. To start fresh for a new day, tap Reset Schedule before uploading a new file.

On Overlay Section

This section labelled On Overlay — Automatic by schedule always shows what is currently displayed on the stream banner. It updates automatically as sessions change. You can also edit it freely at any time.

SAVE CHANGES

Edit the program name or speaker name, then tap this to update what is shown on the stream — without hiding the banner. Great for fixing a spelling mistake while you are live.

Jump to This Speaker

Every session card under the Next Event: label has a Jump to this speaker button.

→

Tap it to immediately show that speaker on the stream banner — useful when a speaker arrives early or you need to switch manually.

→

The schedule keeps running in the background. When the next session's time arrives, it will automatically switch back to the correct speaker.
